# イントロ台本（日本語・簡易版）

## 配置とDB属性
- ファイル配置: `assets/intro/*.wav`
- DB属性: `kind="intro"`, `category="system"`, `source="asset"`
- 推奨尺: 1.5〜3.0秒、トーン: `whisper` / `soft`

## 再生ルール
1. セッション開始時に候補を取得（`getIntroPhrases()`）
2. 1件ランダム選択 → 一度だけ再生 → 通常シーケンスへ

---

## 台本サンプル

### 中性システム（最小）
- 「これから睡眠シャッフルを始めます。聞こえた言葉をそのままイメージしてください。」
- 「睡眠シャッフルを開始します。言葉をただ思い浮かべるだけで大丈夫です。」
- 「準備ができました。始めます。聞こえた言葉をイメージしてみてください。」

### やさしい囁き（soft/whisper）
- 「これから、睡眠シャッフルを始めます。聞こえた言葉を、そのまま思い浮かべてください。」
- 「いまから、やさしく始めます。言葉をただイメージしてみてください。」
